Step 8 — Firestore Security Rules
08
Tighten Firestore Rules
Guided walkthrough + one-tap copy
Guide
1

Open console.firebase.google.com and select your project

2

Left sidebar → Firestore Database → tap the Rules tab

3

Select all existing text and delete it, then paste the rules below

4

Tap Publish — your rules are now live ✓

Firestore Rules 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId}/{document=**} {
      allow read, write: if request.auth != null
                         && request.auth.uid == userId;
    }
  }
}
✅ Logged-in users can only access their own data
❌ No one can read another user's data
❌ Logged-out visitors cannot read anything

Step 7 — Firestore Security Rules (Guided)
07
Firestore Security Rules
Guided + one-tap copy — only logged-in users can access data
Guide
1

Open console.firebase.google.com → tap your project

2

Left menu → tap "Firestore"

3

Tap the "Rules" tab at the top

4

Select ALL existing code and delete everything

5

Paste the rules below using the Copy button

6

Tap "Publish" → tap "Publish" again to confirm

Firestore Rules 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId}/{document=**} {
      allow read, write: if request.auth != null
                         && request.auth.uid == userId;
    }
  }
}
✅ Only logged-in users can read their own Firestore data
✅ Each user can only see their own data, not others
❌ Nobody without login can access any data
